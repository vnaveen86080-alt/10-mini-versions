# Calculator App

A simple calculator built using HTML, CSS, and JavaScript.

## Features

* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Interactive button-based input
* Clear button to reset calculations
* Responsive grid-based calculator layout
* Real-time expression input

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How It Works

1. Users enter numbers and operators using the calculator buttons.
2. The application separates numbers and operators into arrays.
3. JavaScript processes the expression and calculates the result without using `eval()`.
4. The result is displayed in the console/output area.
